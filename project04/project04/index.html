Comparison Operators

Operators Meaning Example and Return Value Explanation of Example
== Is Equal To 'Libby' == 'Libby' returns true. The string values on both sides of the == operator are identical.
=== Strict Equal To 7 === '7' returns false. The numeric value on the left side of the === operator does not match the string value of 7 on the right side. Since these are not the same data type AND the same value, false is returned.
!= Is Not Equal To 5 != '5' returns false. Since the non-strict operator != will automatically take numeric string value and convert it to numeric in JavaScript, the operator understands these two values as being equal, so it returns false.
!== Strict Is Not Equal To 5 !== '5' returns true. Using a strict is not equal to operator will prevent the automatic type conversion and take the values being compared as they are passed. Since the numeric value of 5 and the string value of 5 are not the same, the expression evaluates to true.
> Greater Than 3 > 6 returns false. Because the value of 3 is less than, and NOT greater to 6, false is returned.
>= Greater Than Or Equal To 4 > 4 returns true. The numeric value of 4 is more than or equal to the value of 4. Since this is true, the value of true is returned.
< Less Than 4 < 5 returns true. The numeric value of 4 is less than the numeric value of 5, so true is returned.
<= Less Than Or Equal To 6 <= 3 returns false. The numeric value of 6 is NOT less than the numeric value of 3, so false is returned.
&& Logical And (4 > 2) && (3 < 6) returns true. We know that 4 is more an 2, so the left side of && is true. Since the right side is also true, true is returned. For the logical and, either side of the expression must evaulate to true for a true to be returned. If both side do not equal true, false will be returned.
|| Logical Or (4 < 2) || (3 > 6) returns false. The logical not must have either one of both sides of it's expression evaluate to true. If neither side is true, false is returned.
! Logical Not !(3 === '3') returns true. Since numeric 3 and string value 3 are NOT equal, we get false. The logical not tells us to take that value and make it the oposite, which comes to true.
